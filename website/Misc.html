Miscellaneous resources Overview

This page contains some resources you may want to look at.

ExploitDB

Exploitdb is, as it sounds, a (website) database full of exploits. Found a vulnerability, but need to look for an exploit? Trying to look for one on exploitdb.

On the website, once you find the exploit you’re looking for, go into its page. Within the page, the code for the exploit program can be found, as well as details such as what type of exploit it is, platforms it works on, CVE, whether it’s been verified.

It’s a very good site for looking for and finding exploits. Just make sure you're using the right exploit for the right version of vulnerability if it's relevant.

Kali

Kali Linux is possibly one of the most well knowns virtual machines out there (outside of Ubuntu). This VM is a premade penetration testing environment that comes with an arsenal of tools.

The reason I did not base this guide off a Kali VM is because the UI is not as user friendly as Ubuntu, and it comes preinstalled with many tools I was not going to use (and did not want to confuse anyone when they open the start menu, or take up pointless space).

Regardless, when you become more apt in penetration testing, I definitely recommend taking a look at Kali.

Pen Testing is hard at first, be careful not to do difficult challenges early
