Welcome to Penting!

Hello, this website is designed to provide information regarding common pen testing tools and how you may use them!

The site contains the techniques of each tool, example usage and also dissects the tool to explain things like arguments!

For example, the Nmap page goes into detail about the most common arguments, like -p-, -sU, and how they are used.
When running certain tools (like Nmap) it is very important you keep a look out for case sensitivity. For example, nmap –sn and nmap –sN are two different commands

In addition to this content, I have also added some general penetration testing content and resources, such as HTB, as well as general tips.

Recommended path:

This is the path I recommend following. I recommend this path because some tools are best used before others in practice , and some platforms provide easier tasks

  1. Nmap
  2. Nikto
  3. BurpSuite
  4. BadStore
  5. DVWA
  6. Additional Tasks
  7. Mutillidae (no guide page available, also needs to be setup)
  8. TryHackMe (and then HTB)

I also recommend looking at the Misc Resources page and Legal and Ethics Overview pages I added. They contain helpful information.
